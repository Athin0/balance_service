Приложение преобразует csv и prn файлы в html
---
Название входного и выходного файла задается в main.go в переменных
fileInName - из чего преобразуем 
fileOutName - файл, в который преобразуется

Формат задан в функции MakeFormat4String пакета Formatters
по умолчанию "имя_поля":"данные_поля", по типу json файлов
